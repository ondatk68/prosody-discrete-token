## Benchmarking Prosody Encoding in Discrete Speech Tokens
paper: https://arxiv.org/pdf/2508.11224

Place the TIMIT dataset at the same directory level. 

Run `extract.py` to exract only SX set from TIMIT.

Run `word_level_modification.py` and `utterance_level_modification.py` to modify prosody and vocal tract length via WORLD.
